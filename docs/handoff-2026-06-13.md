  ```bash
  cd livekit-agent && set -a && source ../.env.local && set +a \
    && export CALENDAR_ENDPOINT="https://voice-demo-navy.vercel.app/api/book-meeting" \
    && export WEBHOOK_SECRET="$HUME_TOOL_SECRET" \
    && nohup ./venv/bin/python -u agent.py dev > /tmp/livekit-agent.log 2>&1 &
  ```
  **Nota**: `agent.py` exige `CALENDAR_ENDPOINT` e `WEBHOOK_SECRET` no `os.environ` (linhas 30-31) — não estão em `.env.local`, têm de ser exportados manualmente (valores acima; `WEBHOOK_SECRET` = valor de `HUME_TOOL_SECRET`).
  Se ficar em loop de reconexão: `pkill -9 -f agent.py` e relançar.
